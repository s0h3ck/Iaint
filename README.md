# Iaint

Light methodology to take notes and feel somewhat light :}

## Installation script
```text
pacman -Syu vim texlive-most maim feh evince ctags xclip devmon
sh iaint.sh
```

### Tools
```
* vim
    * lightline.vim
    * NERDTree
    * nerdtree-execute
    * tagbar
* latex
* maim
* feh
* evince
* ctags
* xclip
* devmon
```
